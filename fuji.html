Fuji

Fuji is essentially the "intelligence layer" for autonomous coding agents. While tools like linters find syntax errors, Fuji is a specialized AI Insight Engine designed to help AI agents understand and fix complex architectural issues.

Key Features:

  • Built for Agents: Generates insights in TOON format, a machine-readable structure optimized for LLMs. Uses [ACTIONABLE] tags for direct intervention.
  • 5-Stage Pipeline: Features Static, Symbol, Diff Isolation, AI-Critic, and Supervise stages for thorough codebase analysis.
  • "Kyoto Night" Aesthetic: A custom terminal theme featuring Sakura Pink, Matcha Green, and Wisteria Purple for a beautiful development experience.
